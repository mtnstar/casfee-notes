## Flexbox

```css
display: flex
flex-direction: column, row, row-reverse, column-reverse
justify-content: flex-start, flex-end, ...
```

grow -> es hat zu viel Platz
shrink -> hat zuwenig platz
Default von flex ist: flex: 0 1 auto

## CSS Grid

```css
display: grid;
```
